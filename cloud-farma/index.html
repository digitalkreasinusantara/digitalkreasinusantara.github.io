Aplikasi Pengelolaan Apotek Berbasis Web

Cloud Farma

Cloud Farma adalah aplikasi apotek berbasis web. Selain mudah digunakan, Cloud Farma juga memiliki berbagai fitur sesuai kebutuhan dasar pada manajemen penjualan pada apotek anda.


Fitur:

  • Pengelolaan Data Barang
  • Data Pegawai dan Akses Pegawai
  • Stok Barang dan Penyesuaian Stok Barang
  • Transaksi Pembelian Barang
  • Pembatalan Pembelian Barang
  • Transaksi Penjualan Barang
  • Pembatalan Penjualan Barang
  • Laporan Penjualan
  • Laporan Pembelian
  • Laporan Nilai Aset
  • Laporan Profit

Mengapa harus Cloud Farma?

  • Lebih efisien karena tidak memerlukan server fisik untuk aplikasi, anda hanya perlu gadget & internet untuk mengoperasikannya.
  • Domain dengan nama apotek anda sendiri misal https://namaapotekanda.com.
  • Gratis biaya setup awal & update aplikasi.
